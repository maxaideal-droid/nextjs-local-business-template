# Strong Opinions

These are the only opinions used as editorial takes in blog posts. One per post, maximum. Each opinion must be backed by a specific number from `stats.md` or a cited external source.

An opinion is a direct, non-hedged claim that the brand is willing to defend. Not a preference. Not a gentle suggestion.

---

## Format

```
## [Short title]

**The opinion:** One sentence. Direct. No hedging.

**The backing:** The specific number or external source that supports this claim.

**When to use:** Which post types or topics this applies to.
```
