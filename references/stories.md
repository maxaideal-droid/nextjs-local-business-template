# Recurring Client Stories

These are the only anecdotes used in blog posts and service page copy. One story per post, maximum. Never invent new ones — real stories only.

Add stories here as they happen. Format each one with enough detail that it can be referenced clearly, but scrubbed of any identifying information the client has not consented to share.

---

## How to add a story

```
## [Short identifying title]

**What happened:** One paragraph, plain prose. What the client needed, what happened, what the outcome was. Specific details (day, weather, event type) make it real.

**When to use:** Which types of posts or pages this story fits.

**Do not use when:** Any context where this story would be misleading or out of place.
```
